# Love Calculator

A fun, web-based application that calculates the "chance of love" between two people based on their names.

## 🚀 Features
* **Name Input:** Accepts user and crush names via text fields.
* **Randomized Results:** Uses a logic-based random generator to provide a percentage score from 0% to 100%.
* **Clean UI:** Features a centralized "content-box" with a themed background and responsive layout.
* **Dynamic Feedback:** Displays a personalized message using the names provided once the calculation is triggered.

## 🛠️ Tech Stack
* **index.html**: Defines the structure, including labels, inputs, and a calculation button.
* **style.css**: Manages the aesthetics, including a heart-themed background, whitesmoke container, and custom button styling.
* **script.js**: Handles the DOM event listener for the "Calculate" button and the result logic.

## 📦 How to Run
1. Clone this repository.
2. Ensure you have an image named `hearts.jpg` in the same directory for the background.
3. Open `index.html` in any modern browser.
